# R-Programming-Coursework-SLE251-Research-Methods-Data-Analysis

R programming coursework and statistical data analysis projects completed for SLE251: Research Methods & Data Analysis (June-October 2024), uploaded retrospectively for portfolio and archival purposes.

This project was completed using R, primarily through the R Commander (Rcmdr) interface for data management, statistical analysis, and visualisation. The materials in this repository were originally completed in 2023 and have been uploaded retrospectively to GitHub to document my academic work and provide an accessible portfolio of previous projects.

**Repository Contents**
>R programming worksheets and practical exercises
>Statistical analysis outputs and visualisations
>Scientific report and supporting materials
> Coursework files demonstrating data analysis and interpretation using R and R Commander (Rcmdr)

**Project Overview**
The primary project in this repository investigates the relationship between second-dose measles vaccination coverage and measles incidence across WHO-designated African regions between 2000 and 2022. Using publicly available World Health Organization (WHO) data, the project applies data visualisation, correlation analysis, and linear regression techniques to examine whether increased vaccination coverage is associated with reduced disease incidence.

Through this project, key concepts in research methodology, statistical analysis, hypothesis testing, and scientific communication were applied using R and the R Commander graphical user interface.
